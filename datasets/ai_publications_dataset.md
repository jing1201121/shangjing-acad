# AI Publications Dataset Schema

This dataset encompasses **academic publications** in the field of artificial intelligence from **1995 to 2024**.  
It includes comprehensive metadata on AI-related articles, including author information, institutional affiliations, citations, funding organizations, and more. The dataset serves as a valuable resource for analyzing trends in AI research, author collaboration, and funding dynamics.

---

## Dataset Overview

| Field | Description |
|--------|-------------|
| **Publication Type** | Type of publication (e.g., article, book chapter, conference paper). |
| **Authors** | List of authors for the publication. |
| **Author Full Names** | Full names of the authors (for disambiguation). |
| **Authors ID** | Unique identifier for each author (e.g., ORCID, ResearcherID). |
| **Authors Information** | Additional details about the authors (e.g., academic background, research focus). |
| **Book Authors** | Authors of the book (if applicable). |
| **Book Editors** | Editors of the book (if applicable). |
| **Book Group Authors** | Group authors of the book (if applicable). |
| **Book Author Full Names** | Full names of authors in books (if applicable). |
| **Group Authors** | Authors from a group or consortium (if applicable). |
| **Article Title** | Title of the article or paper. |
| **Source Title** | Title of the journal or book where the article was published. |
| **Book Series Title** | Title of the book series (if applicable). |
| **Book Series Subtitle** | Subtitle of the book series (if applicable). |
| **Language** | Language of the publication. |
| **Document Type** | Type of document (e.g., journal article, conference proceeding). |
| **Conference Title** | Title of the conference where the paper was presented (if applicable). |
| **Conference Date** | Date of the conference. |
| **Conference Location** | Location of the conference. |
| **Conference Sponsor** | Conference sponsor (if applicable). |
| **Conference Host** | Host of the conference (if applicable). |
| **Author Keywords** | Keywords chosen by the authors to describe the research. |
| **Keywords Plus** | Additional keywords from external indexing sources. |
| **Abstract** | Abstract of the publication. |
| **Addresses** | Addresses of the authors (affiliation addresses). |
| **Affiliations** | Institutional affiliations of the authors. |
| **Reprint Addresses** | Reprint address for correspondence. |
| **Email Addresses** | Email addresses of the authors. |
| **Researcher Ids** | Identifiers for researchers (e.g., ResearcherID). |
| **ORCIDs** | ORCID identifiers for authors. |
| **Funding Orgs** | Organizations that funded the research. |
| **Funding Name Preferred** | Preferred name for funding organizations. |
| **Funding Text** | Description or text related to funding. |
| **Cited References** | List of references cited in the publication. |
| **Cited Reference Count** | Number of references cited in the article. |
| **Times Cited, WoS Core** | Citation count from Web of Science Core Collection. |
| **Times Cited, All Databases** | Citation count across all databases. |
| **180 Day Usage Count** | Number of times the publication has been used in the last 180 days. |
| **Since 2013 Usage Count** | Number of times the publication has been used since 2013. |
| **Publisher** | Publisher of the publication. |
| **Publisher City** | City where the publisher is located. |
| **Publisher Address** | Address of the publisher. |
| **ISSN/eISSN** | International Standard Serial Number (ISSN) or Electronic ISSN (eISSN). |
| **ISBN** | International Standard Book Number (ISBN), if applicable. |
| **Journal Abbreviation** | Abbreviation of the journal title. |
| **Journal ISO Abbreviation** | ISO abbreviation of the journal title. |
| **Publication Date** | Date of publication. |
| **Publication Year** | Year of publication (1995-2024). |
| **Volume** | Volume number of the publication (for journals). |
| **Issue** | Issue number of the publication (for journals). |
| **Part Number** | Part number of the publication (if applicable). |
| **Supplement** | Supplement number (if applicable). |
| **Special Issue** | Indicates whether the article is part of a special issue. |
| **Meeting Abstract** | Whether the publication is a meeting abstract. |
| **Start Page** | First page number of the article. |
| **End Page** | Last page number of the article. |
| **Article Number** | Article number (for journals without page numbers). |
| **DOI** | Digital Object Identifier for the publication. |
| **DOI Link** | Direct link to the publication via DOI. |
| **Book DOI** | DOI for the book (if applicable). |
| **Early Access Date** | Date when the publication became available in early access. |
| **Number of Pages** | Total number of pages in the publication. |
| **WoS Categories** | Web of Science categories related to the publication. |
| **Web of Science Index** | Indexing information for Web of Science. |
| **Research Areas** | Research areas covered by the publication (e.g., Machine Learning, NLP, Robotics). |
| **IDS Number** | Identifier number from the publisher's ID system. |
| **PubMed Id** | PubMed identifier (if applicable). |
| **Open Access Designations** | Whether the publication is open access. |
| **Highly Cited Status** | Whether the publication is categorized as highly cited. |
| **Hot Paper Status** | Whether the publication is categorized as a hot paper. |
| **Date of Export** | Date when the data was exported. |
| **UT (Unique WOS ID)** | Unique identifier in the Web of Science system. |
| **Web of Science Record** | Web of Science record identifier. |
| **Year** | Year of the publication. |
| **Contains USA** | Whether the publication includes researchers or institutions from the USA. |
| **Only USA** | Whether the publication only includes researchers or institutions from the USA. |
| **Contains USA and China** | Whether the publication includes both USA and China-based researchers or institutions. |
| **Contains China** | Whether the publication includes researchers or institutions from China. |
| **Only China** | Whether the publication only includes researchers or institutions from China. |
| **Contain Industry** | Whether the publication involves industry collaboration. |
| **Only Industry** | Whether the publication only involves industry collaboration. |
| **Contain Academic** | Whether the publication involves academic collaboration. |
| **Only Academic** | Whether the publication only involves academic collaboration. |
| **Industry-Academic** | Whether the publication involves collaboration between industry and academia. |
